# Globe Snake

A 3D snake game played on the surface of a sphere. Navigate around the globe, eat pellets to grow, and avoid colliding with your own tail.


## Run Locally

This is a static site with no build step. Serve the files with any static server:

**Using Node.js:**
```bash
npx serve
```

**Using Python:**
```bash
python -m http.server
```

Then open http://localhost:3000 (serve) or http://localhost:8000 (Python) in your browser.

## Controls

| Input | Action |
|-------|--------|
| ← / A | Turn left |
| → / D | Turn right |
| Space | Pause / Resume |

You can also tap/click the left or right side of the game canvas to steer.
